# 08 - Writing

Draft and compile the technical report with all sections, figures, and references.

## Required Scripts

- `setup_latex.sh` - Initialize LaTeX template and bibliography
- `compile_report.sh` - Build PDF from LaTeX source
- `extract_code_snippets.py` - Generate code listings for appendix
- `format_references.py` - Process bibliography entries
- `check_spelling.sh` - Run spell checker on report text
- `validate_report.py` - Check for missing figures, references, sections
