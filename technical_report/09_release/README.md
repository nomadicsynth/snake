# 09 - Release

Prepare and execute the release of the technical report, models, code, and announcement materials.

## Required Scripts

- `upload_models_to_hf.py` - Upload trained models to HuggingFace Hub
- `upload_datasets_to_hf.py` - Upload datasets to HuggingFace Hub
- `generate_model_cards.py` - Create model cards for HuggingFace
- `create_github_release.sh` - Tag and release code on GitHub
- `publish_to_website.sh` - Deploy report PDF to neoncortex.com.au
- `generate_announcement.py` - Create social media posts and announcements
- `prepare_youtube_materials.sh` - Export videos and descriptions for YouTube
