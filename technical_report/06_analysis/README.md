# 06 - Analysis

Aggregate results from all experiments, compute statistics, identify patterns, and prepare data for visualization and writing.

## Required Scripts

- `collect_all_metrics.py` - Aggregate metrics from training/eval logs into unified format
- `statistical_analysis.py` - Compute significance tests and confidence intervals
- `identify_failure_modes.py` - Analyze when and why models fail
- `generate_results_tables.py` - Create LaTeX/Markdown tables for report
- `export_for_figures.py` - Prepare data files for figure generation
- `summarize_findings.py` - Generate executive summary of key results
